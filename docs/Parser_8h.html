class  Parser
 Parser is a class for parsing a sequence of tokens into an abstract syntax tree. It contains functions for parsing expressions, statements, declarations, and operators. The parser is used by the interpreter to parse the tokens generated by the lexer. The parser is implemented as a recursive descent parser.
