Detailed Description

AstVisitor is an abstract base class for visiting the nodes in an abstract syntax tree. It contains virtual functions to visit each type of node in the tree, which must be implemented by derived classes. The nodes that can be visited include expressions, statements, declarations, and operators.

Note
This class is part of a larger system for parsing and interpreting a programming language.
