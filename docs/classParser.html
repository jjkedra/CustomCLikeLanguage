Detailed Description

Parser is a class for parsing a sequence of tokens into an abstract syntax tree. It contains functions for parsing expressions, statements, declarations, and operators. The parser is used by the interpreter to parse the tokens generated by the lexer. The parser is implemented as a recursive descent parser. The functions in the class correspond to the grammar rules of the programming language.

Note
This class is part of a larger system for parsing and interpreting a programming language.
See also
Lexer, Token, AstVisitor, Tree.h
